# ED Visit Analysis

## Project Overview
This project involves performing data analysis on the `Emergency_Department_Visits` collection using MongoDB. It includes a variety of queries ranging from basic document counts to complex aggregation pipelines and the creation of a visual dashboard in MongoDB Atlas.

## Data Set
- Source: CDC - [Estimates of Emergency Department Visits in the United States](https://healthdata.gov/CDC/Estimates-of-Emergency-Department-Visits-in-the-Un/i3me-g36r/about_data)
- Collected from 2016 - 2022
- Scope: National Hospital Ambulatory Medical Care Survey (NHAMCS) data featuring metrics such as visit estimates, reliability scores, and demographic subgroups
  
## Tools 
- MongoDB: For data storage and query execution.
- MongoDB Compass / Shell: Used for collection-based, aggregation, and nested querying.
- MongoDB Atlas Charts: For dashboard creation - view [dashboard](https://charts.mongodb.com/charts-final-project-luztuls/public/dashboards/674e586f-0412-45e3-8bba-1386a5d8ad46)

### Key Visualizations Included:
- Average Emergency Visits by Race Per Year 
- Distribution of Emergency Visits by Measure Type 
- Trends in Emergency Visits by Primary Payment Source (2016–2022) 
- Emergency Visit Rates by Age Groups
